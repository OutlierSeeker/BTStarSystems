BATTLETECH Planet Systems README

This utility provides additional options to view and analyze the star systems found in BATTLETECH (2018). You can zoom in the map, choose which planets are to be displayed and then save the map to a PNG file. Furthermore, you may export some planetary data to a convenient CSV file that you can then import into your favorite spreadsheet program.

User Interface

Folder Selection

When you run BATTLETECH Star Systems for the first time, you will presented with a file selection screen. Choose your BATTLETECH main folder (which contains "BattleTech.exe"), e.g. "D:\Games\Steam\steamapps\common\BATTLETECH" like in the following example:

BATTLETECH Star Systems works with any version of BATTLETECH and reads the star systems' JSON files from the data folder.

The Main Application

The Left Panel

In the left panel you can set the attributes of the planets that you want to display. The left radio button column is the default setting and means that the attribute may or may not be present on a star system. The middle column means that the attribute has to be present while the column means that the attribute may not be present (exclude).

The numbers after each attribute mean how many star systems have this attribute within the current selection. In the screenshot above, all attributes are at default and so for this version of BATTLETECH, there are a total of 219 star systems.

The Star Map

The middle section displays the star map with all systems that fulfill the set attributes. It also has a menu button which expands/contracts when clicking on it.

  • Zooming: The zoom buttons should be self explanatory and are the main reason why I created this program (I wanted a nice overview of the whole map instead of only parts of it. When zooming out, the systems may overlap, especially when additional information is displayed.
  • Show Skulls: Displays the skull rating next to each star system.
  • Show Black Markets: Indicates which system has a black market.
  • Show Star League: Shows if a planet has a former Star League presence.
  • Show max. shop items in side panel: Does exactly that.
  • Show planet description in side panel: I added this just for fun. The description is usually pretty long and doesn't add anything useful, but it might still be interesting for some.
  • Save Map: Saves the current star system configuration with the current zoom level to a PNG file. This opens a new dialog where you can tweak some settings, but the aforementioned settings are applied to the export, i.e. a zoomed in map will created a larger file than a zoomed out one.

    Most settings should be self explanatory. If "Draw Faction Colors" is unticked, the system names will be black or white, depending on the "Draw White Background" setting (which I included for easier printing).
    The default output directory is the current directory where BATTLETECH Star Systems is located and will automatically rename the default name if a file already exists (e.g. "map1.png", "map2.png", etc.). However, you may manually select a file name and choose to override any existing files.
  • Opten BT folder: With this buttong you can change your BATTLETECH folder in case you need it (i.e. should you have more than one installation).

The Right Panel

On the right you have a list of all star systems that fulfill the current set of attributes. Hovering the mouse over a planet name will expand the panel to show a more detailed view of the system. Furthermore, the star system is highlighted on the map.

The INI File

After running the program for the first time, an INI file is created automatically for some additional options. While most of the info saved there for convenience reasons (GUI and last saved folder), there are two settings that might be of interest:

  • calculateStats: Running BATTLETECH Star Systems with this set to "true" will create a CSV file that contains the owner, employer and target faction statistics of all loaded planets. While this information is also available in the GUI, I found it more convenient to have the data separately in order to adequately analyze the data and plan my next career route.
  • directDirectorySelection: By setting this to true you choose the folder that contains the star system JSON files directly instead of just selecting the main BATTLETECH folder. WARNING!!! Make sure that you select the right folder (by either entering the right path in the INI file or deleting the old path from lastDir and then selecting the path through the dialog), otherwise the program will not work correctly.
  • useLog: This is basically a debug option. Setting this to "true" will then create a "log.html" when running the program which displays some info about the program.
